# student-management-system-mysql
Student Management System using MySQL: Designed a relational database with tables for students, courses, enrollments, and marks. Implemented constraints, complex SQL queries (joins, aggregations), views, and stored procedures to manage and analyze academic data efficiently.

# Student Management System – Documentation
Introduction: The Student Management System is a relational database project developed using MySQL. It is designed to store and manage student details, courses, enrollments, and marks efficiently. The system demonstrates database design, data manipulation, and advanced SQL operations used in real-world applications.

# Database Design
Tables Used:
Students
Courses
Enrollments
Marks

Explanation:
Students → Stores student details
Courses → Stores course information
Enrollments → Links students and courses
Marks → Stores student performance
